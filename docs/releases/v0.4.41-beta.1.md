## Purpose

Promote the post-`v0.4.40-beta.1` outcome-observation follow-up as a named
local-worker beta. PR `#373` deepens scheduled outcome collection so the daemon
can detect revert, hotfix, merged-fix, and human-thread signals without leaving
the live worker on untagged `main`.

## Included Changes

- Expand scheduled outcome observation signal handling for revert, hotfix,
  merged-fix, and human-review-thread evidence.
- Add bounded GitHub file reads and resolved-value window caching in the
  outcome fetch path.
- Tighten revert-matching behavior and add focused coverage for the new outcome
  scenarios.

## Required Gates
