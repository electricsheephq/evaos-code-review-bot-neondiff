# v0.4.21-beta.1

## Outcome

Ships the merged public-product sprint batch from #118, #123, and #232:
deterministic metadata pre-merge gates, supported addon dry-run packets for repo
wiki/GitNexus context, and the NeonDiff public beta package release-readiness
surface.

## Highlights

- Added deterministic pre-merge metadata checks with configurable section
  headings, placeholder handling, min-detail guidance, and focused tests.
- Added supported addon dry-run packet contracts for OpenWiki-style repo wiki
  packets and GitNexus context packets.
- Preserved GitNexus integration while making redaction status honest:
  GitNexus packets now report `unknown` unless an explicit redaction status is
  supplied.
- Added public release-readiness workflows, install script, packlist/claim/secret
  checks, and npm publish workflow auth wiring.
- Assigned and closed #232 under the v1.0 public MVP milestone.
