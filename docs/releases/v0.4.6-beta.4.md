release PR jobs that were already in the durable queue before the priority
hotfix could retain background priority `50` and sit behind ordinary backlog.

## Changes

- Reprioritize existing queued and provider-deferred automatic
  `electricsheephq/evaos-code-review-bot` jobs before budget and lease
  selection.

  preserves provider cooldown metadata, leases, review URLs, and terminal
  fields.

  need to be leased and retired ahead of ordinary backlog when the PR has
  already closed or merged.
