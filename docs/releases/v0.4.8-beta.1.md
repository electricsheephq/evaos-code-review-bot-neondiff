## Purpose

Promote the merged NeonDiff Desktop cleanup work from PR `#186` and the
Saiba-inspired desktop theme work from PR `#187` as a named beta release so the
live launchd worker is not running an untagged main checkout. The release also
records the exact-head gate caveat from the out-of-band merges: both PRs had
green GitHub checks and no unresolved review threads, but their final heads
merged before evaOS recorded terminal review proof for those exact SHAs.

## Changes

- Harden NeonDiff Desktop CLI resolution so directories are not accepted as
  executables and GUI-safe user-bin paths are shared across launch and PATH
  construction.
- Restrict the desktop subprocess environment to a small allowlist plus a
  controlled GUI-safe `PATH` instead of copying the full inherited environment.
- Redact all present config secret values in public config output, including
  empty string values, while preserving `undefined` and `null` shape.
- Document nested JSON patch shape for desktop config writes and clarify that
  flat dotted keys are rejected.
- Add focused regression coverage for config redaction, desktop config patch
  safety, and CLI launch behavior.
- Apply a Saiba-inspired black/neon-green operator theme to the NeonDiff
  Desktop SwiftUI development MVP, including shared theme tokens, angular
  panels, grid/scanline surfaces, themed sidebar/detail views, and status
  badges.
- Keep supplied Saiba assets as visual reference only; no third-party font file
  is bundled in this release.
