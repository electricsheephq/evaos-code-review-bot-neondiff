## Purpose

Promote the public NeonDiff pricing contract into the source beta. This release
adds a deterministic `neondiff pricing` JSON command and aligns README, setup,
license-boundary, pricing docs, and public tests around the decided support
tiers: public open-source repository review is free; paid support is $1/month,
$10/year, or $100 lifetime.

## Included Changes

- Adds `neondiff pricing`, a network-free JSON pricing and entitlement-vocabulary
  command for Free OSS, Monthly Support, Yearly Support, and Lifetime Support.
- Adds `docs/pricing.md` as the canonical public support-tier reference.
- Aligns README, setup docs, license-boundary wording, and license/setup issue
  template placeholders with the pricing doc.
- Locks the BYOK/local-provider boundary in tests: NeonDiff pricing does not
  include hosted model credits, unlimited SaaS inference, or bundled provider
  tokens.
- Fixes the public CLI provider-deferred queue fixture so its synthetic active
  provider lease is relative to the test process clock instead of a stale
  absolute date.

## Caveats

- This release does not add payment processing, hosted model credits, or license
  API billing.
- This release does not change live daemon repo allowlists, launchd config, or
  provider behavior.
- Runtime license enforcement is outside the pricing command; this release
  defines the pricing and entitlement vocabulary that enforcement code and docs
  should use.
