## Purpose

Promote the private-repo entitlement proof matrix and fail-closed license-gate
coverage added for `#326`. This release proves the worker blocks private or
commercially gated repos before checkout, file listing, provider calls, or
review posting when a required entitlement is missing or not active.

This release does not publish a new npm package, widen runtime permissions,
change monitored repos, enable issue enrichment for additional repos, or change
review model behavior. It is a source/local-worker beta release while
`neondiff@0.4.30-beta.1` remains the current public npm package.

## Included Changes

- Add a direct license-gate matrix for public, private, and unknown repo
  visibility across missing, active, expired, and revoked entitlement states.
- Prove that private active entitlements cover both private and public-gated
  repos, while public-only entitlements do not cover private repos.
- Prove the worker-level private-repo gate returns `skipped_license_gate`
  before checkout, pull-file listing, provider execution, or review posting.
- Make revoked entitlements report `revoked` before expiry so operator evidence
  explains the actual non-active entitlement state.
- Update setup and license-boundary docs with provider-present and
  provider-absent license gate proof expectations.
